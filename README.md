# RNAmicroarray-ADdataset

“Developed a pipeline for transcriptomic microarray datasets' analysis to identify Alzheimer’s disease-related pathways”

Overview

Brain tissue microarray data (GSE5281) were obtained from Gene Expression Omnibus

Methods

Single-modality analysis • Normalization and preprocessing
• Differential abundance (transcriptomics)

Motivation

This project was developed to build expertise in transcriptomics datasets and demonstrate the ability to combine heterogeneous biological datasets
